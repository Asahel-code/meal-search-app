# Meal Search application

This is a web application build by vue js and intergrated with a public api to search for a meal of you choice

### Project directory structure

Project directories are structured as follows:

```
root
    ├── /.github
        ├──/workflows
    ├── /public
    ├── /src
        ├── /assets
        ├── /components
        ├── /composables
        ├── /router
        ├── /views
        ├── /api
        ├── /types
index.html
netlify.toml
package.json
package-lock.json
postcss.config.ts
tailwind.config.ts
tsconfig.app.json
tsconfig.app.tsbuildinfo
tsconfig.json
tsconfig.node.json
tsconfig.node.tsbuildinfo
vite.config.ts
CHANGELOG.md
README.md
```

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app services in the development mode.\
Open [http://localhost:5173](http://localhost:5173).

### `npm run build`

Bundle the app to production mode.
